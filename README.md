I kajal Barthwal build a full-stack Markdown blog application that allow users to create,update and delete blog using Markdown with Authentication and role-based access.


User Authentication (Register / Login / Logout)

Role-based access (Admin & User)

Create, Update, Delete Blogs

Image upload support

Markdown-based content

Responsive UI

Protected Dashboard

Profile management

Separation of concerns (controllers, routes, middleware)

-----------Tech Stack ------------

1) Frontend

React.js – Component-based UI

React Router DOM – Client-side routing

Tailwind CSS – Responsive and modern styling

Axios – API communication

React Hot Toast – User notifications

React Hot Toast

2) Backend

Node.js & Express.js – REST API

MongoDB & Mongoose – Database & data modeling

JWT Authentication – Secure auth with cookies

Cloudinary – Image upload & storage





--------Design Decision----------


Context API used for global authentication and blog state management

Role-based access control (Admin/User) for secure operations

Reusable components for scalability and maintainability

Protected routes to prevent unauthorized access

Cookie-based JWT authentication for improved security

Responsive design using Tailwind CSS for all screen sizes

Clean separation of frontend and backend for better architecture






-----------Prerequisites----------

Node.js installed

MongoDb installed or MongoDB compass account

git installed


This project demonstrates a modern full-stack architecture, strong fundamentals in React, backend APIs, authentication, and real-world application design.
